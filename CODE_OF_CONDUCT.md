# 行为准则 Code of Conduct

## 我们的承诺

为营造开放友善的环境，作为贡献者与维护者，我们承诺让参与本项目与社区的体验对所有人都无骚扰，不论年龄、体型、可见或不可见残障、族裔、性别认同与表达、经验水平、教育、社会经济状况、国籍、外貌、种族、宗教或性取向。

本准则采用 [Contributor Covenant](https://www.contributor-covenant.org/) v2.1 为基础。

## 期望的行为

- 展现对他人的同理心与善意
- 尊重不同的观点、经历与立场
- 接受建设性反馈，并给予同等
- 承担自身言行的责任，并向受影响者致歉
- 以社区整体利益为重，而非个人

## 不可接受的行为

- 使用性化语言或图像，或以任何性挑逗/追求
- 嘲讽、侮辱、贬低性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可，发布他人的私人信息（住址、邮箱等）
- 在专业场合被合理认为不妥的其他行为

## 执行

若遇到违规行为，请通过 [SECURITY.md](./SECURITY.md) 所列渠道私下联系维护者报告。所有投诉都会得到及时、公平的审查与处理。

维护者有权也有责任删除、编辑或拒绝与本准则不符的评论、提交、代码、wiki 编辑、issue 及其他贡献，并在必要时临时或永久封禁任何贡献者。

## 适用范围

本准则适用于所有项目空间（issue、PR、讨论区、外部沟通如邮件/社群），以及任何以本项目官方身份公开代表项目的场合。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/)，版本 2.1，完整文本见 https://www.contributor-covenant.org/version/2/1/code_of_conduct.html
